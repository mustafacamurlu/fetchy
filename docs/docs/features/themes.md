# Themes & Customization

Fetchy offers extensive visual customization to match your preferences.

---

## Built-in Themes

9 themes are available out of the box:

| Theme | Description |
|-------|-------------|
| **Dark** | Deep dark background, high contrast |
| **Light** | Clean white/gray interface |
| **Ocean** | Deep blues and teals |
| **Forest** | Muted greens and earth tones |
| **Earth** | Warm browns and amber hues |
| **Aurora** | Northern lights inspired pastels |
| **Flame** | Warm oranges, pinks, and purples |
| **Candy** | Bright, playful pastels |
| **Rainbow** | Vibrant multi-color accents |

Switch themes from **Settings → Theme**.

---

## Custom Themes

Create a fully custom theme with your own color palette:

1. Open **Settings → Theme**
2. Click **+ Create Custom Theme**
3. Configure colors:

| Color Slot | Purpose |
|-----------|---------|
| **Background** | Main app background |
| **Sidebar** | Sidebar background |
| **Card** | Panel/card backgrounds |
| **Text** | Primary text color |
| **Border** | Panel and element borders |
| **Accent** | Buttons, links, highlights |
| **+ more** | Additional slots for fine-tuning |

4. Enter a theme name and click **Save**

### Editing & Deleting Custom Themes

- Saved custom themes appear alongside built-in themes
- Click the **edit** icon to modify a custom theme
- Click **delete** to remove it

---

## Panel Layout

Switch the response panel between two layouts:

| Layout | Description |
|--------|-------------|
| **Horizontal** (default) | Request on left, response on right |
| **Vertical** | Request on top, response below |

Change via **Settings → Panel Layout** or the toggle in the toolbar.

---

## Resizable Panels

- Drag the **divider handle** between the sidebar and main area to resize the sidebar
- Drag the **divider handle** between the request and response panels to adjust the split

---

## Collapsible Sidebar

Click the **collapse** button (or use the sidebar toggle) to hide the sidebar completely, maximizing the workspace area for the request/response panels.

---

## Syntax Highlighting

All code editors and response viewers use **CodeMirror 6** for syntax highlighting, supporting:

- JSON
- YAML
- JavaScript
- TypeScript
- HTML / XML
- Plain text

---

## Variable Highlighting

Variables in URLs, headers, and body fields are highlighted inline:
- 🟢 **Green border/underline** — variable exists in active environment
- 🟡 **Yellow border/underline** — variable is not defined
