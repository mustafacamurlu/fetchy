# Keyboard Shortcuts

Press **`Ctrl+/`** at any time to view shortcuts in the app.

---

## Request Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Send current request |
| `Ctrl+S` | Save current request |
| `Ctrl+N` | Create new request tab |
| `Ctrl+W` | Close current tab |

---

## Collections & Import

| Shortcut | Action |
|----------|--------|
| `Ctrl+I` | Open Import dialog |
| `Ctrl+E` | Open Environments modal |

---

## App Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+/` | Show keyboard shortcuts help |

---

## Tips

- **`Ctrl+N` then type URLs rapidly** — fastest way to prototype multiple API calls
- **`Ctrl+Enter`** works from anywhere in the request panel — no need to click Send
- **`Ctrl+S`** after tweaking a request saves without navigating away
- Use numeric shortcuts (`Ctrl+1-9`) to quickly jump between open request tabs
