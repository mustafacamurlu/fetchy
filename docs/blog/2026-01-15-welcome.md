# Welcome to the Fetchy Documentation Site

We are thrilled to launch the official Fetchy documentation — your comprehensive guide to the privacy-first REST API client.

<!-- truncate -->

## What is Fetchy?

Fetchy is a **100% local, self-hosted REST API client** for developers who value:

- 🔒 **Privacy** — your API keys, credentials, and request data never leave your machine
- 🌐 **Offline capability** — full functionality without an internet connection  
- 🏠 **Data ownership** — all data stored as plain JSON files in directories you control

> Local by design. Reliable by nature.

### Features
Deep dives into every feature:
- HTTP request builder with all methods and body types
- Authentication (Bearer, Basic, API Key) with inheritance
- Collections, folders, and the Collection Runner
- Environments and variable substitution with `<<variable>>`
- Pre/post-request JavaScript scripting and assertions
- Code generation for 8 languages
- Import from Postman, OpenAPI, and cURL
- Workspaces with isolated secret storage
- 9 built-in themes + fully custom theme editor

Practical, step-by-step walkthroughs for real-world scenarios.
